Simple Single-Page-Application pager

This is an easy to use / hack / extend JavaScript pagination.
It is non intrusive and based on URL hash (example.com/_#my-hash_). It simply uses the native HTML anchor.

See installation and usage for further details.

Oh, what you are currently browsing is based on the SPAger.js!

How to use the spager.js?

Using the spager is very easy:

  1. import _spager.js_
  2. add a class=page in your HTML Nodes that should behave like pages
  3. add an id attribute to each; that will be the page name.

Now when you want to display a page just call the hash matching the page name. Just like an HTML anchor!

This current page is done with spager.js. Just view the source for an example or checkout the basic demo.
